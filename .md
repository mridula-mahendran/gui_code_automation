# OpenText UFT Flight GUI Test Automation

## Description
Automated test script for the OpenText MyFlight Booking Application using UFT (Unified Functional Testing).

## Features
- Data-driven framework (reads from Excel/DataTable)
- 4 iterations with different flight bookings
- 4 checkpoints (3 pass, 1 intentional fail)
- Screenshots captured before and after each form
- Object Repository with manually added objects

## Test Data
- 4 different flight bookings with different cities, classes, and passengers
- Login credentials: John / HP

## Tools Used
- OpenText UFT One CE 26.1
- VBScript
- Windows Task Scheduler
